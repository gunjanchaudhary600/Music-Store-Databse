## Music Store SQL Analysis Project

# Project Overview
- This project focuses on solving SQL queries using a Music Store dataset.
- The main objective of this project is to practice and demonstrate SQL skills from Basic to Advanced level queries.
- I have solved different types of queries to analyze customer behavior, sales performance, and music trends.

# Tools & Technologies Used

- SQL (PostgreSQL / MySQL)
- Music Store Database

# Dataset Description
- The dataset contains information about:
- Customers, Invoices, Invoice Items, Artists, Albums, Tracks, Genres, Employees
- This dataset helps in understanding sales patterns and customer purchasing behavior in a music store.

# SQL Concepts Covered
- Basic Level Queries: SELECT statements, WHERE conditions, ORDER BY, LIMIT, Simple JOIN, Aggregate functions (COUNT, SUM, AVG)
- Moderate Level Queries: GROUP BY, HAVING clause, Multiple JOINs, Subqueries, Filtering with conditions
- Advanced Level Queries: Complex JOIN operations, Nested Subqueries, CTE (Common Table Expressions), Window Functions, Revenue and customer insights analysis

# Key Insights Generated
- Top selling artists and genres
- Highest revenue generating countries
- Most valuable customers
- Sales trends analysis
- Employee performance insights

# Project Objective
- The goal of this project is to strengthen SQL skills and build a strong foundation in data analysis using structured query language.

This project helped me improve my understanding of: Data extraction, Data aggregation, Business problem solving using SQL, Writing optimized queries
